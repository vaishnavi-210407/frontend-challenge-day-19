# 📝 Blog UI

A responsive Blog UI built using HTML, CSS, and JavaScript.

## 🚀 Features
- Dynamic blog rendering
- Responsive card layout
- Blog title and description display
- Read More button
- Clean modern UI

## 🛠 Tech Stack
- HTML
- CSS
- JavaScript

## 📂 Project Flow
1. Created blog data array
2. Rendered blog cards dynamically
3. Added responsive layout
4. Implemented Read More button
5. Styled clean blog interface

## 🎯 Learning Outcome
- Improved dynamic rendering skills
- Better understanding of UI layouts
- Practiced responsive card design
- Learned structured content display
